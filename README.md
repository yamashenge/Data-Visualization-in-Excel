Sales Performance Dashboard (Excel)

Overview
This project features a Sales Performance Dashboard built in Microsoft Excel to visualize monthly sales across three regions for the year 2025. The dashboard provides an interactive and clear summary of sales performance, helping businesses make data-driven decisions.

Problem Statement
Many businesses struggle to:
Identify sales trends over time.
Determine top-performing regions.
Recognize seasonal patterns in sales.

Without proper visualization, these insights are often hidden in raw data, making strategic planning difficult.

Solution
Our dashboard addresses these challenges by:

Presenting monthly sales data in an easy-to-understand format.
Highlighting regional performance through charts and pivot tables.
Allowing interactive filtering using slicers for deeper analysis.

This enables businesses to quickly spot trends, compare regions, and plan effectively.

Tools & Techniques

Microsoft Excel for data analysis and visualization.
Pivot Tables & Charts for summarizing and comparing data.
Slicers for interactive filtering.
Basic Data Cleaning using Excel functions.


Features

Monthly sales visualization for 2025.
Comparison across three regions.
Interactive filters for dynamic analysis.
Clear and professional dashboard layout.


How to Use

Download the Excel file from this repository.
Open in Microsoft Excel (2016 or later recommended).
Use slicers to filter by region or month.
Explore charts and pivot tables for insights.
